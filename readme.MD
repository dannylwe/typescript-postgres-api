# Start Database
run `npm run database`

# start application
clone repo  
`cd typescript-postgres-api`  
run `npm install`  
run `npm run dev`

# Connect to Postgres Docker Container
run `docker exec -it [postgres-container-name] -U postgres`

# How to setup Postgres on Docker
visit Docker hub.

# Postgres Commands
- show databases: \l
- use database: \c
- show tables: \dt
- describe table: \d tablename
